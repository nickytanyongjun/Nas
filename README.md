# Nas
This project is a fully functional, self‑hosted Personal Cloud (NAS) web application built with Python (FastAPI) for the backend and a single HTML file with vanilla JavaScript and CSS for the frontend, designed to expose your entire D:\ drive (Or any drive that u want) as a secure, browsable file explorer that works from any browser on your local network or anywhere in the world when paired with a tunneling tool like Cloudflare or ngrok. The interface mimics a native file manager, allowing you to navigate folders by clicking on directory names, preview supported file types (including images, PDFs, text files, videos, and audio) directly in your browser without forced downloads, and download any file or even entire folders—which are automatically zipped on the server before being sent to you. For uploading, the intuitive click‑based dropdown menu separates file and folder uploads, and the backend correctly preserves the complete folder structure when you upload a whole directory, while robust path‑sanitization prevents any directory traversal attacks. The UI also features checkbox‑based multi‑selection for safely deleting multiple items at once (with a confirmation step), a smart “Back” button that only appears when you are inside a subfolder, and a fully custom modal system that replaces every browser alert with sleek, theme‑matching popups for all success or error messages, ensuring a polished and consistent experience. To get started, you only need Python 3.8 or higher; install the required packages (fastapi, uvicorn, and python-multipart), change the default admin password directly inside main.py for security, and run python main.py—the terminal will display both your local 127.0.0.1:8000 address and your network IP so you can access it from other devices on the same WiFi, while for remote internet access, simply run cloudflared tunnel --url http://localhost:8000 alongside the server to get a public HTTPS link. This project combines the simplicity of a single‑file deployment with the performance of FastAPI’s asynchronous streaming, making it an ideal lightweight, open‑source alternative to commercial cloud storage for anyone who wants full control over their data without sacrificing ease of use.
